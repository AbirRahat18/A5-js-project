Help Center FAQs

What are the different ways to select an element in the DOM?
getElementById(): Selects an element by its id.

getElementsByClassName(): Selects elements by class name (returns a live HTMLCollection).

getElementsByTagName(): Selects elements by tag name (returns a live HTMLCollection).

querySelector(): Selects the first element matching a CSS selector.

querySelectorAll(): Selects all elements matching a CSS selector (returns a static NodeList).
What is the difference between innerHTML, innerText, and textContent ?
innerHTML: Includes HTML tags.
innerText: Includes visible text only.
textContent: Includes all text, even if hidden.
What is event delegation in the DOM?
Event delegation means instead of adding event listeners to each child element individually, you add a single event listener to a parent element, and catch events from its children using event bubbling. It saves memory, improves performance, and handles dynamically added elements easily.
What is event bubbling in the DOM?
Event bubbling means when an event happens on a child element, it first runs its own handler, then the event moves up (bubbles) to its parent, then grandparent, and so on up to the root
How do you create, add, and remove elements using JavaScript?
Create element: Create a new HTML element using JavaScript. (createElement) Add element: Insert the created element into a parent element. (appendChild or append) Remove element: Delete an existing element from the DOM. (remove)
